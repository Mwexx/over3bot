# Download and Import

## Import into Deriv Bot Builder

1. Open `https://app.deriv.com/bot`.
2. Go to **Bot Builder**.
3. Click **Import**, choose **Local**, and select:

   `exports/deriv-over3-bot.xml`

You can also drag the XML file directly onto the Deriv Bot Builder workspace.

## Download to another device

Use the package created at:

`dist/deriv-over3-package.zip`

That ZIP contains the Python bot, the Deriv-style dashboard, and the importable XML file.

## Phone note

Phones can download and upload the XML file, but the full Python bot needs a Python-capable environment. For normal phone use, import the XML into Deriv Bot in the browser.
